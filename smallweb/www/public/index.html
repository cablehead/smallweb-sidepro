Creating a site / app / domain with Smallweb is as simple as creating a directory. This installation hosts two domains, www and example. You can view the source for both here.

Out of the box, you also get webdav. cli. apps, which you can use to manage your entire Smallweb installation. To make use of these you'll need to generate a token, which you can do using SidePro's exec feature. You only need to exec once, after that, you can use the cli. endpoint to manage tokens, as well as most everything else.
